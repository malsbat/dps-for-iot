Parameters
nodeThe node that received the data
endpointThe endpoint that received the data
statusIndicates if the receive was successful or there was a network layer error
dataThe raw data
lenLength of the raw data
Returns
- DPS_OK if the message was correctly parsed
  • An error code indicating the data received was invalid
